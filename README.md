# CodeAlpha_LanguageTranslationTool

## About
A Language Translation Tool built with Python 
and Flask that translates text between 20+ languages 
using MyMemory Translation API.

## Features
- Translate between 20+ languages
- Copy translated text instantly
- Beautiful and responsive UI
- Supports English, Hindi, Tamil, Spanish,
  French, German, Japanese, Korean and more

## Technologies Used
- Python
- Flask
- MyMemory Translation API
- HTML, CSS, JavaScript

## How to Run
1. Clone the repository
2. Install requirements:
   pip install flask requests
3. Run: python app.py
4. Open: http://127.0.0.1:5000

## Project Structure
- app.py - Flask backend
- templates/index.html - Main interface
- static/style.css - Styling
- static/script.js - Frontend logic
